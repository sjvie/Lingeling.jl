Lingeling.jl Documentation

This is Lingeling.jl, a Julia package providing an interface to the SAT solver Lingeling created by Armin Biere (Website, Repo).

Windows support

Lingeling.jl does currently not work on Windows.

While supporting the most important functions of Lingeling, this package does not include access to the full Lingeling API. For any functionality beyond what is provided here, please call the Lingeling C API directly using Lingeling_jll. For reference, see the Lingeling.jl source code.
